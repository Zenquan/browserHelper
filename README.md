# browserHelper
## 背景
> 工作中需要使用一些辅助的工具，但是迫于以下原因，就决定自己使用vue写一个浏览器插件。
- 1、一个工具就装一个插件，最后装了n个；
- 2、有的工具用起来，不大符合自己的习惯、体验；
- 3、插件的扩展性不够好；
